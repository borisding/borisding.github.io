Here is a brief of my personal projets worked in my spare time. Feel free to check it out if you are keen to know more about it.

1. Avenue (Repo)

Home-bake PHP MVC framework for rapid web prototyping and development. The idea behind of this framework is inspired by Kohana HMVC framework.

  • Support PHP 5.5 and above.
  • PSR standards (PSR-2, PSR-4).
  • Few base components (Cookie, Session, Validation, etc).
  • Simple Street ORM, built on top PDO.
  • Fast, lightweight and extensible.

2. Avenue-Express (Repo)

A starter that allows developers to implement typical server-side MVC design in Node Express framework. The routing idea is basically borrowed from the Avenue project.
