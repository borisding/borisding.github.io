Here is a brief of my personal projets worked in my spare time. Feel free to check it out if you are keen to know more about it.

1. Avenue (GitHub Repo)

Home-bake PHP MVC framework for rapid web prototyping and development. The idea behind of this framework is inspired by Kohana HMVC framework. Basically, I initiated this project due to sick of the frequent change and non backward compatible of framework APIs out there. Also, take this project as a challenge to myself and craft it as what I want it to be.

2. Avenue-Express (GitHub Repo)

A starter that allows developers to implement typical server-side MVC design in Node Express framework. The routing idea is basically borrowed from the Avenue project.
